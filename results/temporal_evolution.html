Temporal Evolution Analysis

Watch how embedding models have evolved from 2021 to 2025 through four animated visualizations. Explore performance trends, technology adoption patterns, efficiency improvements, and innovation leadership across developers and countries. Use the play/pause controls and timeline sliders to navigate through time.

1. Performance Evolution Timeline Animated

Trace the journey of model performance improvements over time. Each bubble represents a model, sized by parameter count, colored by developer. Watch the timeline unfold to see how MTEB/BEIR scores have improved year over year.

2. Technology Adoption Timeline Animated

Visualize when key technologies emerged and gained adoption. See the spread of Matryoshka embeddings, quantization strategies, architectural innovations, and feature support across the timeline. Darker colors indicate more models using each technology.

3. Efficiency Evolution Animated

Track how models became more efficient over time. The red dashed line shows the "efficiency frontier" - models that achieved the best performance for their parameter count. Watch as this frontier shifts upward, showing continuous improvement in the performance-to-size ratio.

4. Innovation Race: Developer & Geographic Leadership Animated

Watch the innovation race unfold as different developers and countries compete to release high-quality embedding models. See which organizations and regions led the field at different points in time, and how leadership patterns shifted from 2021 to 2025.

  • Year Selection: Use the dropdown to select any year (2021-2025) - shows ONLY models from that specific year
  • Color By: Change visualization colors across all charts - choose from Developer, Performance (MTEB/BEIR), Year, Parameter Count, or Efficiency
