Comprehensive Performance Heatmap

Interactive multi-view heatmap showing RAG performance, external benchmarks, technical specifications, and efficiency metrics across all embedding models. Use the dropdown menu to switch between different metric categories. Values are normalized within each category for easy comparison.

How to Use This Visualization

  • Select Metric Category: Use the dropdown menu above the chart to switch between RAG Performance, External Benchmarks, Technical Specs, Efficiency metrics, or view all metrics combined
  • Hover for Details: Hover over any cell to see the actual metric value and normalized score
  • Zoom & Pan: Click and drag to zoom into specific areas, double-click to reset
  • Reset View: Use the reset button in the modebar (top-right) to restore the default view
  • Download: Click the camera icon to download the current view as PNG
  • Normalization: Values are normalized (0-1 scale) within each metric group for fair comparison across different scales
