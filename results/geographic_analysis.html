How to Use This Analysis

  • Tabs: Switch between different analysis views using the tabs above
  • Filters: Use the left sidebar to filter data by year, country, model size, or developer
  • World Map: Switch between different views (model count, average accuracy, benchmark scores) using the dropdown menu
  • Hover for Details: Hover over any chart element to see detailed information about models, countries, and metrics
  • Interactive Controls: Use zoom, pan, and selection tools on each chart for detailed exploration
  • Timeline: Explore how model releases have evolved over time across different countries
  • Developer Ecosystem: Understand which organizations are developing models in each country
  • Technical Analysis: Compare technical specifications (architecture, dimensions, licenses) by country
  • Benchmarks: Analyze how models from different countries perform on various benchmark suites
  • Download: Click the camera icon on any chart to save it as an image
